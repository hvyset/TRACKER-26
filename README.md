# TRACKER 26 — Deploy Guide

## What you need
- A free GitHub account (github.com)
- A free Vercel account (vercel.com) — sign in with GitHub

---

## Folder structure
Set up your files like this:

```
tracker26/
├── public/
│   ├── manifest.json
│   ├── sw.js
│   ├── icon-192.png     ← make a simple icon (see below)
│   └── icon-512.png     ← same icon, bigger
├── src/
│   ├── main.jsx
│   └── tracker26.jsx
├── index.html
├── package.json
└── vite.config.js
```
